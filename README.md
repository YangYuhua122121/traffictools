# traffictools
 A toolkit for undergraduate transportation students  
 一个向交通本科生提供的python工具包  
  
 该工具包有如下特色：  
 1. 纯中文文档，而且讲解得还算清楚吧(lll￢ω￢)
 2. 涵盖规划、设计、仿真，以及一些第三方库与工具的快速入门
 3. 基于Pandas，可读性和可操作性好，可高效处理大规模数据
 4. 开源、易获得、免翻墙、免费、普通本科生友好型
